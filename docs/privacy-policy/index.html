Privacy Policy

Salesforce Debug Lens

Thank you for using Salesforce Debug Lens, a Chrome extension designed to help Salesforce developers debug Apex code more efficiently by understanding and visualizing Salesforce debug logs.

Your privacy is important to us. This Privacy Policy explains what data the extension accesses, how it is used, and how we protect your information.

Information We Collect

Salesforce Debug Lens does not collect, store, or transmit any personal information to external servers or third parties. All data is used locally within the extension for its intended functionality.

  • Session ID from Salesforce cookies: Used to authenticate API calls to your connected Salesforce org.
  • Organization Information: Retrieves org name and type (Production or Sandbox) for identification and context.
  • User Information: Fetches display name and ID of the logged-in user from Salesforce.
  • Debug Logs: Fetched on demand to analyze and visualize code execution, errors, and flow.
  • Apex Classes and Triggers: Metadata is retrieved based on log insights for relevant execution flow.
  • Trace Flags & Debug Levels: Manage log generation via Salesforce Tooling APIs.

How We Use Your Data

All data accessed by Salesforce Debug Lens is processed and used only within your local browser environment. No data is sent to any external servers. The data is used exclusively to:

  • Display organization and user context.
  • Parse and visualize debug logs for Apex code analysis.
  • Manage Salesforce trace flags and debug levels.

Data Storage

  • Local Storage: Minimal org data (e.g., org name and type) is stored locally to improve user experience.
  • No External Storage: No data is stored in any cloud service or remote database.

You can clear the local data at any time via your browser’s settings.

Data Sharing

We do not share, sell, or transfer any data accessed by the extension to any third parties under any circumstances.

User Consent

By using Salesforce Debug Lens, you acknowledge and agree that:

  • You have the right and permission to access the Salesforce org you're connecting to.
  • You understand the extension only uses your data to provide debug-related functionality.

Your Rights and Choices

No Personal Data Collected:

As no personal data is collected or stored, there are no additional user rights to manage or delete personal data.

If you have questions or require clarification about our data handling practices, you may contact us directly.

Security

We follow industry best practices to ensure that your data is handled securely. Since all processing occurs locally, no external risks of data exposure are introduced by this extension.
